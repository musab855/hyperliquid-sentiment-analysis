# Hyperliquid Trader Sentiment Analysis

## Objective
Analyze the relationship between Bitcoin market sentiment and trader performance.

## Datasets
- Bitcoin Fear & Greed Index
- Hyperliquid Historical Trader Data

## Key Findings
- Most trading activity occurred during Fear.
- Highest average profit occurred during Greed.
- Extreme Greed reduced profitability.
- Market sentiment affects trader behavior and outcomes.

## Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
